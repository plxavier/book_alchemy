# 📚 Book Alchemy

A Flask-based digital library app to manage books and authors.

## Features

- Add, view, edit, and delete books
- Add and view authors
- Search and sort books
- Dashboard with library stats
- JSON API endpoints

## Tech Stack

- Python 3
- Flask
- SQLAlchemy
- SQLite
- Jinja2 HTML templates

## Setup

### 1. Clone the repo
- git clone https://github.com/plxavier/book_alchemy.git
- cd book_alchemy

### 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate

### 3. Install dependencies
pip install -r requirements.txt

### 4. Run the app
python app.py


## Dependencies

- Flask==2.3.2
- Flask-SQLAlchemy==3.1.1
- Flask-Migrate==4.0.4
